JavaScript - Funções, Arrays e Objetos

Funções

// Uma função é um bloco de códigos designado para realizar uma tarefa. function oProduto(n1, n2) {   return n1 * n2; // Retorna o produto de n1 com n2 } // Uma função é executada quando é invocada por alguém. oProduto(2, 5); // Isso retornará 10
// Podemos atribuir uma função anônima a uma variável. var nomeFuncao = function() {   console.log ("Olá"); } nomeFuncao();

Escopo (Scope)

O escopo de uma variável é o contexto em que ela é visível.

function myFunction() {   var a = 4; // a é uma variável local   return a * a; } var a = 4; // a é uma variável global function myFunction() {   return a * a; }

Funções Aninhadas

Podemos ter uma função dentro de outra função.
A função aninhada será visível apenas dentro da função pai.

function circunferencia (raio) {   // função aninhada   function diametro() {     return 2*raio; // raio é uma variável da função pai   }   return Math.PI * diametro(); // invocamos a função }

Exemplo

function a(){   console.log('a vem primeiro'); } function b(){   console.log('b vem depois'); } a(); b();
function a(){   setTimeout(function(){     console.log('a vem depois');   }, 500) } function b(){   console.log('b vem primeiro'); } a(); b();

Callbacks

Callback é uma função passada como parâmetro a outra função,
que será executada após o término da primeira.

JavaScript é uma linguagem assíncrona.
Isso significa que é possível executar uma invocação sem saber quando ela termina.
Para poder administrar essa situação, utilizamos o padrão de design callback.

Os callbacks podem ser usados de várias formas.
No próximo exemplo, utilizamos um callback com uma função anônima.

function a( callback ){   setTimeout(function(){     console.log('a vem primeiro');     callback();   }, 1000) } function b(){   console.log('b vem depois'); } a( b );

Arrays

Arrays - Método forEach

[1, 5, 7].forEach(function(value, index) {   console.log("No índice: " + index + " está o valor: " + value); });

Arrays - Método map

O map é usado para modificar cada um dos itens de um array
utilizando uma função determinada.

[1, 5, 7].map(function(numero) {   return numero * 2; }); // O resultado será uma nova array com os valores [2, 10, 14]

Arrays - Método filter

O filter é usado para filtrar alguns elementos de um array.

[13, 18, 20].filter(function(numero) {   return ( numero >= 18 ); }); // O resultado será uma nova array com os valores [18, 20]

Arrays - Método reduce

O reduce é usado para reduzir um array a um valor único,
utilizando uma função determinada.

[1, 5, 7].reduce(function(total, numero) {   return total + numero; }); // O resultado é um valor único: 13

Objeto literal

Definição de Objetos

Um objeto é qualquer item do dia a dia representado em códigos.

var carro = {   marca: "Chevrolet",   modelo: "Corsa",   quilometragem: 65000,   cor: "Branco",   donos: ["Luciana", "João", "Zezinho"] };

Os objetos têm propriedades usadas para descrevê-lo.
Elas podem ser de vários tipos (number, string, array, object, etc.)

Propriedade de Objetos

É possível acessar uma propriedade de um objeto de várias formas:

carro.marca; // "Chevrolet" carro["marca"]; // "Chevrolet"

É possível acessar utilizando variáveis:

var prop = "marca"; carro[prop]; // "Chevrolet"

É possível alterar o valor de uma propriedade:

carro.cor = "Vermelho"; carro["cor"] = "Vermelho";

Também é possível adicionar propriedades novas a um objeto:

carro.velocidadeMax = 200; carro["velocidadeMax"] = 200;

Métodos de Objetos

Também existem métodos para interagir com os objetos.
Um método é uma propriedade do objeto com uma função atribuída.

var carro = {   marca: "Chevrolet",   ligar: function(){ console.log("vrum vrum"); },   modelo: "Corsa" }; carro.ligar(); // É possível utilizar carro["ligar"]();

Como são funções, os métodos também podem receber parâmetros.

var carro = {   marca: "Chevrolet",   motorista: function(nome){     console.log(nome + " está dirigindo");   },   modelo: "Corsa" }; carro.motorista("Pedro");
