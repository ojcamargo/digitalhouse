DOM, seletores e elementos

DOM (Document Object Model)

DOM - Definição

O document object model (DOM) oferece uma representação
estrutural de um documento html.

Com o DOM, o JavaScript consegue:

  • Modificar elementos, atributos e estilos de uma página.
  • Excluir qualquer elemento e atributo.
  • Adicionar novos elementos ou atributos.
  • Responder a todos os eventos HTML da página.
  • Criar novos eventos HTML na página.

SELETORES

Seletores

Para acessar os elementos de uma página, usamos seletores.
É possível acessar usando os nós, mas isso pode ser um pouco confuso.
Cada seletor pode retornar apenas um elemento ou um array.

O objeto document tem os seguintes
seletores como método:

  • document.getElementById(ID)
  • document.getElementsByClassName(classe)
  • document.querySelector(cssQuery)
  • document.querySelectorAll(cssQuery)

document.getElementById()

<html> <head> </head> <body>   <h1 id="cabecalho">Bem-Vindo!</h1>   <script>     document.getElementById("cabecalho").style.display = 'none';   </script> </body> </html> // getElementById seleciona o elemento com o id informado.

document.getElementsByClassName()

<html> <head> </head> <body>   <h1 class="cabecalho">Bem-Vindo!</h1>   <h2 class="cabecalho">Bem-Vindo, meu amigo!</h2>   <script>     let elementos = document.getElementsByClassName("cabecalho");     console.log(elementos); // HTMLCollection(2) [h1.cabecalho, h2.cabecalho]     for(var i=0; i<elementos.length; i++){       elementos[i].style.color = 'green'     }   </script> </body> </html> // getElementsByClassName seleciona todos os elementos que possuem a classe informada.

document.querySelector()

<html> <head> </head> <body>   <h1 class="cabecalho">Bem-Vindo!</h1>   <script>     document.querySelector(".cabecalho").style.color = 'blue';   </script> </body> </html> // querySelector() seleciona o primeiro elemento que cumprir a condição.

document.querySelectorAll()

<html> <head> </head> <body>   <h1 class="vermelho">Bem-Vindo!</h1>   <h2 class="vermelho">Bem-Vindo, meu amigo!</h2>   <script>     let elementos = document.querySelectorAll(".vermelho");     console.log(elementos); // HTMLCollection(2) [h1.vermelho, h2.vermelho]     for(var i=0; i<elementos.length; i++){       elementos[i].style.color = 'red'     }   </script> </body> </html> // querySelectorAll seleciona todos os elementos que cumprirem a condição.

Atributos

Assim que um elemento é selecionado, é possível acessar
os atributos dele usando a propriedade attributes.
Ela retorna um mapa (objeto/array) que tem valores key/value
com os nomes e valores dos atributos desse elemento.

let atributos = elemento.attributes;

getAttribute()

O método getAttribute aceita uma string como parâmetro com o nome
do atributo que queremos obter. Retorna o valor do atributo.
Caso ele não seja encontrado, retorna null.

let href = elemento.getAttribute("href");

setAttribute()

O método setAttribute permite adicionar um novo atributo
ou modificar um existente.

elemento.setAttribute("class","vermelho");

hasAttribute()

O método hasAttribute aceita uma string como parâmetro com o nome do
atributo que queremos saber se existe no elemento.
Retorna um valor booleano.

elemento.hasAttribute("href");

removeAttribute()

Com o método removeAttribute, podemos remover um atributo existente.

elemento.removeAttribute("class");

Estilos

  • Os elementos HTML têm uma propriedade chamada style,
    que retorna um objeto literal que representa os estilos desse objeto.
  • É possível adicionar ou modificar seus atributos.
  • Os nomes das propriedades CSS em JavaScript são escritos
    no seguinte formato: nomeDaPropiedadeDeCss.
elemento.style.color = "red"; elemento.style.fontWeight = "bold";

ELEMENTOS

Criando Elementos

O método createElement permite criar novos elementos HTML.
createElement aceita como parâmetro strings com nomes de tags HTML
(a, div, span, li, ul, etc).

let btn = document.createElement("BUTTON");

O método createTextNode permite criar novos textos HTML.

let texto = document.createTextNode("Olá, sou um texto!");

Inserindo elementos - appendChild

O método appendChild permite inserir um nó(elemento) dentro de outro.

<body>   <ul id="myList"></ul>   <script>     let li = document.createElement("LI");     let liText = document.createTextNode("Water");     li.appendChild(liText);     document.getElementById("myList").appendChild(li);   </script> </body>

textContent

O método textContent permite ler ou escrever conteúdo como texto.
Quando ele lê, ele remove todas as tags HTML do conteúdo.

<body>   <h1 id="cabecalho"><strong>Bem-Vindo!</strong></h1>   <script>     let elemento = document.getElementById("cabecalho");     console.log(elemento.textContent); // "Bem vindo!"     elemento.textContent = "Novo texto!";     console.log(elemento.textContent); // "Novo texto!"   </script> </body>

innerHTML

O método innerHTML permite ler ou escrever conteúdo em HTML.

<body>   <h1 id="cabecalho"><strong>Bem-Vindo!</strong></h1>   <script>     let elemento = document.getElementById("cabecalho");     console.log(elemento.innerHTML); // "<strong>Bem vindo!</strong>"     elemento.innerHTML = "<b>Novo texto!</b>";     console.log(elemento.innerHTML); // "<b>Novo texto!</b>"   </script> </body>

Removendo elementos - removeChild

O objeto nó(node/elemento) tem um método chamado removeChild
que permite remover nós(elementos) filhos.
Para poder remover um nó, primeiro precisamos selecioná-lo.

<body>   <div id="pai">     <div id="filho"></div>   </div>   <script>     let pai = document.getElementById("pai");     let filho = document.getElementById("filho");     pai.removeChild(filho);   </script> </body>
