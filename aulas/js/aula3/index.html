JavaScript integrado a HTML

Adicionar JavaScript a um arquivo HTML

  1. Inserindo o código na tag <script>.
<body>   <!--Página HTML-->   <script>     // código JavaScript   </script> </body>
  1. Vinculando a um arquivo externo:
<body>   <!--Página HTML-->   <script src="arquivo.js"></script> </body>

Para depurar JavaScript na linha de comando do navegador, utilizamos:

console.log()

<body>   <script>     console.log(5+6);   </script> </body>

alert()

O método alert permite mostrar uma mensagem na tela,
com apenas 1 botão "OK" para fechar.
Esse método deve ser usado apenas para informar o usuário.

<script>   alert("ALERTA VERMELHO!"); </script>

prompt()

O método prompt permite interagir com o usuário,
pois podemos pedir que ele insira um valor.

<script>   var nome = prompt("Insira seu nome:"); </script>

A resposta inserida pelo usuário será salva na variável nome.

confirm()

O método confirm mostra uma mensagem para o usuário,
com 2 botões: "SIM" e "NÃO" ou "OK" e "CANCELAR",
e retorna um booleano(true/false).

<script>   var maior = confirm("Você é maior de idade?"); </script>

A resposta inserida pelo usuário (true ou false)
será salva na variável maior.

Objetos:
window e document

window X document

O objeto window é o primeiro que é carregado no navegador.

O objeto window tem propriedades como length, innerWidth, etc..

O objeto document representa o html, php ou outro documento que será
carregado no navegador. O document é carregado dentro do objeto window
e tem propriedades como title, url, cookie, etc..

window X document

window

Para acessar as propriedades de window, usamos window.nomePropriedade
ou, para facilitar, apenas nomePropriedade.

window.innerHeight // retorna a altura da janela do navegador window.innerWidth // retorna a largura da janela do navegador window.open() // abre uma nova janela window.close() // fecha uma janela

Todos os métodos como alert, confirm, etc. pertencem a window.
Sua notação é opcional, ou seja, pode ser feita das duas formas:

window.alert() == alert() window.innerHeight == innerHeight // Inclusive as variáveis globais fazem parte de window: var teste = 123; console.log(window.teste) // Imprimirá 123 no console.

location
Location é um objeto que representa a URL.
Por meio das propriedades dele, é possível acessar várias partes da URL,
além de navegar para outro documento.

location = 'http://www.google.com'; //redireciona para google.com console.log("A URL atual é: "+ location.href ); location.host; // retorna o nome de host e o número de porta. location.pathname; // mostra o path do recurso location.search; // mostra os parâmetros da query string

window.onload
O window.onload é uma função do Javascript que é chamada
quando a página termina de carregar completamente.
Inclusive imagens, css e scripts. Veja um exemplo abaixo:

window.onload = function() {   console.log("Pronto, página carregada completamente!"); }

document

Para acessar as propriedades de document, usamos document.propriedade

document.body // retorna o body do html document.readyState // Retorna o status do carregamento do documento document.write() // Escreve expressões HTML com JavaScript document.title // Retorna o título da página

document.onload
O document.onload é uma função do Javascript que é chamada
quando a página termina de carregar completamente.
Porém antes de carregar as imagens e similares.
Veja um exemplo abaixo:

document.onload = function() {   console.log("Pronto, página carregada!"); }
