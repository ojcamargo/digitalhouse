Beleza, mas como eu começo a estruturar o código?

Bom, a estrutura pode variar dependendo da sua necessidade. Mas podemos começar escolhendo o container que queremos usar!

Se você quer utilizar um container que utiliza toda a largura da tela, utilize o .container-fluid, caso queira um container com o conteúdo centralizado utilize o .container

Tem mais detalhes na documentação.

E como funciona o grid?

Grid é o esquema que o bootstrap usa para estruturar a página. Não é nada muito complexo, quando você entender, vai facilitar muito seu trabalho com CSS.

O espaço é divido em linhas e colunas, sendo a soma de colunas sempre dá 12.

Para adicionar uma linha basta utilizar a classe .row

Tem um padrão de nomenclatura das classes de colunas no bootstrap.

Começa sempre com .col- depois tem o tamanho da tela lg, md, sm ou xs, e adicionar quantas colunas o seu elemento vai ocupar, sendo de 1 à 12.
Exemplo: col-lg-12

Se você precisa de 2 caixinhas do mesmo tamanho, crie 2 caixas com espaço de 6 colunas.

.col-xs-6
.col-xs-6

Quero que essas caixas oculpem todo o espaço da linha no celular, para o usuário visualizar melhor o conteúdo.

Nesse caso, eu adiciono classes para telas menores, .col-sm-12

.col-lg-6 .col-sm-12
.col-lg-6 .col-sm-12
Mais detalhes na documentação

Exemplo de formulário

O Bootstrap disponibiliza algumas classes para usarmos nos formulários.

Para agrupar campos do formulário é só utilizar a classe .form-group numa div englobando os campos.

E para padronizar os campos, adicione a classe .form-control na tag input.
